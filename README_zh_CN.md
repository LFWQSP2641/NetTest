# NetTest

NetTest 是一个支持 TCP、UDP 和 WebSocket 协议的网络测试工具。

## 功能

- **TCP 支持**：测试和调试 TCP 连接。
- **UDP 支持**：测试和调试 UDP 连接。
- **WebSocket 支持**：测试和调试 WebSocket 连接。
- **DNS 支持**：测试 DNS 查询
